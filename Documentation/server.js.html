Sends an AJAX request to a url of our choice as either a POST or GET

Params
type string
  • Set to either "POST" or "GET" to indicate the type of response we want
url string
  • The URL to send the request to
[success] function
  • A function to run if the call succeeds
[error] function
  • A function to run if the request errors out
params string
  • An "&" delimited string of "key=value" pairs that can be sent as AJAX
Returns
AJAXRequest
  • The request that was sent
